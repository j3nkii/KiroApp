# Product Overview

KiroApp is a work order management system designed for businesses that handle both services and products. The application manages customer information, work orders, and line items with pricing.

## Core Features

- Customer management (contact information)
- Work order creation and tracking
- Line item management with SKU and pricing
- Support for both service and product types
- Price caching at order creation time

## Business Domain

The system appears to be built for service-based businesses that need to track work orders, manage inventory/services, and maintain customer relationships. The database schema supports flexible pricing and order management workflows.
